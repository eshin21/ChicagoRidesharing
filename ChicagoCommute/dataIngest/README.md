# Data Ingest

This folder includes all the necessary files for querying and ingesting the data.

## Data Citations

There are three data sources:

* The United States Census Bureau. “American Community Survey 5-Year Data (2009-2017),” December 6, 2018. https://www.census.gov/data/developers/data-sets/acs-5year.html.

* “Boundaries - City | City of Chicago | Data Portal.” Accessed December 7, 2019. https://data.cityofchicago.org/Facilities-Geographic-Boundaries/Boundaries-City/ewy2-6yfk.

* Levy, Jonathan. “Transportation Network Providers - Trips | City of Chicago | Data Portal,” October 21, 2019. https://data.cityofchicago.org/Transportation/Transportation-Network-Providers-Trips/m6dm-c72p.


## API Keys

API keys from Socrata and the Census Bureau are required. The links and steps for getting those keys are in the ingest files, but here are the direct links for generating a key:

[Socrata API key account signup](https://opendata.socrata.com/login)

[Census Bureau API key request](https://api.census.gov/data/key_signup.html)
